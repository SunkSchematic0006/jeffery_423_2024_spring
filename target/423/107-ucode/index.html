CSE 107 Sample Translations to Unicon and Ucode

These translations are meant to give guidance and provoke questions. They may have bugs or be incomplete/incorrect, especially on details where Python and Unicon semantics differ. They rely on a draft PunY runtime system:
  • puny.icn (.icn), a puny runtime system version 0.1
For any test shown here, if the .u is missing you can generate it by running unicon -c file for the test named file.icn.

Status codes: G=good test, C=compile error, R=runtime error (under Python)
