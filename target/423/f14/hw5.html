Notes

  • your executable should still be named "120"
  • your program should accept and process an arbitrary number of source filenames on the command line
  • take in files with .cpp extensions and write out corresponding final code in files with an appropriate (e.g. .s or .c) extension.
  • write out the name of the file to standard out when you open it
